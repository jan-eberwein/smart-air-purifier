# 🌬️ Smart Air Web Documentation 
How-To Tutorial to rebuild the system

## 🚀 Tech Stack
- 🛠️ **Vite**
- ⚛️ **React**
- 🎨 **Tailwind CSS v4.1**
- 🔗 **React Router** (if necessary)

## 🔧 Setup

1. **Clone & install**  
   ```bash
   git clone https://github.com/your-org/smart-air-web.git
   cd smart-air-web
   npm install
   npm run dev
   ```
   http://localhost:5173
